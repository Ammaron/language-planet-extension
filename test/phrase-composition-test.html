Test 1: ADJ + NOUN (should reorder)

I saw a red cat sitting on the fence yesterday.

Expected: "red cat" → "gato rojo" (ADJ+NOUN reorder, confidence 0.9)

Test 2: ADJ + NOUN (another pair)

The big dog was running in the park.

Expected: "big dog" → "perro grande" (ADJ+NOUN reorder)

Test 3: NOUN + VERB (should preserve order)

The cat sleeps all day on the couch.

Expected: "cat sleeps" → "gato dormir" (NOUN+VERB, preserve order)

Test 4: VERB + NOUN (should preserve order)

We eat food together every evening.

Expected: "eat food" → "comer comida" (VERB+NOUN, preserve order)

Test 5: Solo word (should NOT form phrase)

I put some water in my bag and then grabbed a book from the shelf.

Expected: "water" → "agua" and "book" → "libro" as separate single replacements (too far apart to group)

Test 6: Three-word with glue (ADJ + NOUN + VERB)

The small cat sleeps on the mat.

Expected: "small cat sleeps" → composed phrase (ADJ+NOUN+VERB → NOUN+ADJ+VERB = "gato pequeño dormir", confidence 0.75)

Test 7: Words separated by glue words

A good book is hard to find.

Expected: "good book" grouped through "a" glue word → "libro bueno" (ADJ+NOUN reorder)

Test 8: Complex sentence (should defer to backend)

The big red cat eats good food and sleeps in a small house near the water.

Expected: Multiple phrase groups detected. Some may compose locally, others defer to backend. Check DevTools network tab for PHRASE_TRANSLATE requests.

Test 9: No vocab words (should be untouched)

The weather is sunny today and I went to the store.

Expected: No replacements — none of these words are in the test vocabulary.

Console Inspection

Open DevTools and run these to inspect matcher state:

// Check if GrammarRules loaded
console.log(window.GrammarRules);

// Check what the matcher found (need access to matcher variable)
// The matcher is scoped inside content.js, but you can inspect DOM:
document.querySelectorAll('.lp-vocab-phrase').forEach(el => {
  console.log({
    original: el.dataset.original,
    phraseType: el.dataset.phraseType,
    source: el.dataset.source,
    confidence: el.dataset.confidence,
    text: el.textContent,
  });
});

// Check individual word replacements
document.querySelectorAll('.lp-vocab-word').forEach(el => {
  console.log({
    original: el.dataset.original,
    translation: el.dataset.translation,
    wordId: el.dataset.wordId,
  });
});
